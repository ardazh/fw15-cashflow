## Getting Start
### 1. Clone the project

```bash
  git clone https://github.com/arthuriski29/fw15-chiperpay-
```

## 2. Go to the project directory

```bash
  cd fw15-chiperpay-
```

### 3. Run the development server:

```bash
npm run dev
```
